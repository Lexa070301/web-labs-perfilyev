В основе работы приложения лежит компонент vue-kanban
Для работы со времнием и его форматирования для вывода испольуется компонент moment 
Для модального окна применяется компонент vue-js-modal
Список исполюзуемых метадов:

updateBlock - вызывается при перемещении карточки из одной колонки в другую, обновляет статус карточки и пересчитывает количество карточек в колонках

delete_card - удаляет карточку, после чего пересчитывает количество карточек в колонках

confirm - отвечает за перемещение карточек между колонками путём нажания на галочку, после чего пересчитывает количество карточек в колонках

add_card - добавляет карточку в первую колонку с соответствующим описанием, после чего пересчитывает количество карточек в колонках

show - открывает модальное окно с формой редактирования карточки

hide - закрывает модальное окно с формой без сохранения изменений

apply - отвечает за сохранение изменений из формы редактирования в карточке, проверяет валидность введённых данных, в конце пересчитывает количество карточек в колонках

change_theme - отвечает за смену темы
